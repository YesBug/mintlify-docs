# 产品定价

> 按实际用量付费，计价方式灵活多样，满足不同业务规模。

## 计费货币

平台以\*\*美元（USD）\*\*作为计费单位，定价与各官方厂商保持一致。

## 计费方式

平台支持以下三种计费模式，根据模型特性自动适用：

| 计费方式 | 适用场景 | 说明 |
| --- | --- | --- |
| **按 Token 计费** | 文本、Embedding 等 | 分别统计输入/输出 Token 数量计费，缓存 Token 另有优惠 |
| **按次计费** | 图像生成、音频转写等 | 每次请求固定费用，与输出内容量无关 |
| **按秒计费** | 部分视频、音频生成模型 | 按生成内容的时长计费 |

## 提供商倍率与折扣

每个提供商有其独立的**价格倍率**，最终实际价格 = 模型标准价格 × 提供商倍率。

不同等级的用户，在相同提供商下享受的折扣可能不同。详情请参考 [用户等级](/cn/guides/level-group) 页面。

## 充值与支付

- 平台支持通过 **Stripe 信用卡**在线充值
- 企业用户如需其他付款方式或大额充值，请联系我们

## 余额预警

建议开启余额预警，避免因余额不足中断业务：

进入 **工作台 → 个人设置 → 订阅管理**，设置预警金额。当余额低于预警值时，平台将通过邮件/站内信及时通知您。

## 大用量合作

如果您的用量较大，希望获得更高并发上限或专属折扣，欢迎[联系我们](mailto:hi@maxapi.ai)洽谈。

## 退款政策

如需退款，请联系客服，注意以下事项：

- 退款后如用户等级下降，将扣除超出权益部分金额
- 退款将收取至多 **5%** 手续费
- 退款原路退回至原付款账户
