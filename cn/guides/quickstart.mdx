# 平台介绍

> MaxAPI AI 是一站式 AI 模型聚合平台，通过单一 API 接口接入 OpenAI、Anthropic、Google 等数十家主流厂商的最新模型。

## MaxAPI AI 是什么

MaxAPI AI 是面向开发者和企业的 AI 模型聚合服务平台。

只需一个 API Endpoint 和一个 Key，您即可访问来自 OpenAI、Anthropic、Google、DeepSeek 等主流厂商的最新模型，无需分别注册账号、管理多套密钥、单独充值。平台兼容 OpenAI 和 Anthropic **双协议**标准，几乎无需修改现有代码即可完成接入。

## 特色功能

### 一站式模型聚合

只需创建一个 API Key，使用一套统一的 API 标准，即可调用不同供应商、不同模型。告别多平台注册、多密钥管理、多钱包充值的繁琐流程。

- **统一身份管理**：一个 API Key 管理所有供应商的访问权限
- **统一计费体系**：透明的用量统计和成本控制，集中式账户管理
- **丰富的模型选择**：覆盖 OpenAI、Anthropic、Google、DeepSeek 等主流厂商的最新模型

### 双协议原生支持

完整兼容 OpenAI 和 Anthropic 两大协议标准，无缝集成到 Claude Code、Cursor、Cline 等主流开发工具中。根据项目需求自由切换协议，无需担心模型供应商差异。

- **OpenAI 兼容**：通过 OpenAI 标准 API 统一调用平台所有模型
- **Anthropic 兼容**：通过 Anthropic 标准 API 统一调用平台所有模型，完美支持 Claude Code 等工具

### 高容量与高可用

我们为每个主流模型储备了充足的调用容量，确保业务不受单一供应商容量限制的影响。

- **多供应商支持**：重要模型引入多家供应商节点
- **自动故障转移**：当某个供应商出现问题时，系统自动切换到其他可用节点，服务不中断
- **高并发保障**：企业级并发容量，满足大规模产品的调用需求

### 智能路由

系统根据令牌配置，自动从可用提供商中选择最合适的节点处理请求，支持按**延迟、价格、吞吐量**等策略排序，也可以通过 Header 对单次请求临时指定路由，整个过程对代码完全透明。

### 全球边缘节点加速

依托 **Cloudflare 企业版**基础设施，平台在全球部署了分布式边缘节点，无论用户身在何处，请求都会被路由到最近的节点处理，显著降低网络延迟，提升响应速度。

- **全球覆盖**：边缘节点覆盖全球各主要地区
- **低延迟保障**：就近接入，减少跨洋传输延迟
- **高可用架构**：多节点冗余，保障服务稳定性

### 可观测性

从多个维度深入了解 AI 模型的使用情况，快速定位问题、优化成本。

- **详细调用日志**：完整记录每次 API 调用的请求与响应详情
- **用量统计**：实时监控 Token 消耗和调用频次
- **成本分析**：按模型、时间维度分析成本分布
- **性能监控**：追踪响应时间、可用率等关键指标

### 灵活计费

按实际用量付费，无最低消费门槛。支持按 Token、按次、按秒等多种计费方式，随业务规模弹性扩展。

## 支持的接口类型

| 类型 | 说明 |
| --- | --- |
| **文本** | Chat Completion、Embedding、Rerank，同时兼容 OpenAI 和 Anthropic 协议 |
| **图像** | 文生图、图像编辑，支持 Midjourney、Flux、DALL-E 等 |
| **语音** | TTS 文字转语音、STT 语音识别 |
| **视频** | Kling（可灵）、Sora、Luma、Runway 等主流视频生成模型 |
| **音乐** | Suno 等音乐生成模型 |
