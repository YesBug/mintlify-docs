# 快速开始

> 5 分钟完成接入，立即体验 ePhone AI 平台上的所有模型。

## 第一步：注册账号

前往 [ePhone AI 控制台](https://platform.ephone.ai) 注册账号。

注册完成后，您将需要以下两个配置项：

| 变量         | 值                       |
| ---------- | ----------------------- |
| `BASE_URL` | `https://api.ephone.ai` |
| `API_KEY`  | 您在控制台创建的 API 令牌         |

## 第二步：创建 API Key

进入 **工作台 → API 令牌**，点击「添加令牌」，创建成功后系统将展示 API Key。请复制并妥善保存。

## 第三步：发起第一个请求

将 `API_KEY` 替换为您刚才创建的令牌，`BASE_URL` 已固定为 `https://api.ephone.ai`。

<CodeGroup>
  ```python Python theme={null}
  from openai import OpenAI

  client = OpenAI(
      base_url="https://api.ephone.ai/v1",
      api_key="API_KEY",
  )

  response = client.chat.completions.create(
      model="gpt-4o",
      messages=[{"role": "user", "content": "你好！"}]
  )

  print(response.choices[0].message.content)
  ```

  ```javascript Node.js theme={null}
  import OpenAI from "openai";

  const client = new OpenAI({
    baseURL: "https://api.ephone.ai/v1",
    apiKey: "API_KEY",
  });

  const response = await client.chat.completions.create({
    model: "gpt-4o",
    messages: [{ role: "user", content: "你好！" }],
  });

  console.log(response.choices[0].message.content);
  ```

  ```bash cURL theme={null}
  curl https://api.ephone.ai/v1/chat/completions \
    -H "Authorization: Bearer API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "model": "gpt-4o",
      "messages": [{"role": "user", "content": "你好！"}]
    }'
  ```
</CodeGroup>

## Anthropic 协议接入

如果您使用 Claude Code、Cursor 或其他基于 Anthropic SDK 的工具，可以直接使用 Anthropic 兼容接口：

<CodeGroup>
  ```python Python theme={null}
  import anthropic

  client = anthropic.Anthropic(
      base_url="https://api.ephone.ai/anthropic",
      api_key="API_KEY",
  )

  message = client.messages.create(
      model="claude-opus-4-5-20251101",
      max_tokens=1024,
      messages=[{"role": "user", "content": "你好！"}]
  )

  print(message.content[0].text)
  ```

  ```bash cURL theme={null}
  curl https://api.ephone.ai/anthropic/v1/messages \
    -H "x-api-key: API_KEY" \
    -H "anthropic-version: 2023-06-01" \
    -H "Content-Type: application/json" \
    -d '{
      "model": "claude-opus-4-5-20251101",
      "max_tokens": 1024,
      "messages": [{"role": "user", "content": "你好！"}]
    }'
  ```
</CodeGroup>
